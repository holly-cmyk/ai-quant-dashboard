# AI Quant Dashboard

Interactive quantitative dashboard for AI industry chain portfolio analysis.

**Prices auto-update every weekday after US market close** via GitHub Actions.

## Features
- 33 tickers across 8 sectors
- 6 strategies: Momentum, Golden Cross, Equal Weight, Industry Rotation, MVO, Sector Rotation
- Plotly interactive charts
- Risk metrics, drawdown analysis, rolling Sharpe ratios
- Live strategy tracking panel
